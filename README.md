# Week 1 - Basic Data Exploration and Cleaning using Pandas

## Objective
Learn Python basics and perform basic data exploration and cleaning using Pandas.

## Tasks Performed

- Loaded CSV dataset into Pandas DataFrame
- Explored dataset using head(), tail(), shape, columns, info(), describe()
- Identified and handled missing values
- Filtered rows and selected columns
- Removed duplicate records
- Created derived columns (quantity and total_amount)
- Saved cleaned dataset as CSV

## Dataset
Combined_dataset.csv

## Tools Used
- Python
- Pandas
- NumPy
- Jupyter Notebook
